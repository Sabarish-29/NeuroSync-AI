## Design Guidelines

### Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary | #0f1018 | Backgrounds, main text |
| Accent | #00bfff | Highlights, numbers, CTAs |
| Success | #00ff88 | Checkmarks, positive metrics |
| Warning | #ff9500 | Optional EEG callouts |
| Error | #ff3b30 | Comparison negatives |
| Neutral | #8e8e93 | Supporting text |

### Typography

- **Headers:** Helvetica Bold, 48-72pt
- **Subheaders:** Helvetica Medium, 32-40pt
- **Body:** Helvetica Regular, 24-28pt
- **Captions:** Helvetica Light, 18-20pt

### Layout Principles

- Minimal text per slide (max 3-4 bullets)
- High visual contrast
- Large, readable fonts (viewable from back of room)
- One main idea per slide
- Progressive disclosure (reveal complexity gradually)

### Recommended Tools

- **PowerPoint/Keynote** -- most common, export as PDF backup
- **Google Slides** -- cloud-based, collaborative
- **Canva** -- professional templates, free tier sufficient
